# Primeiros Passos

1. **Clone o repositório:**
    ```bash
    git clone https://github.com/seu-usuario/LojaAfiliados.git
    cd LojaAfiliados
    ```

2. **Instale as dependências:**
    ```bash
    npm install
    ```
    ou, se estiver usando yarn:
    ```bash
    yarn
    ```

3. **Rode o projeto com Vite:**
    ```bash
    npm run dev
    ```
    ou:
    ```bash
    yarn dev
    ```

4. **Acesse no navegador:**
    ```
    http://localhost:5173
    ```
